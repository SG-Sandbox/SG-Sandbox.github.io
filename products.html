How it Works

  1. Noticeable language barrier.
  2. Press "ON" button on EchoBot.
  3. Say phrase into robot.
  4. The speech recognition system will recognize an audio input and convert it to text.
  5. Our customized code will identify the language of input text. This utilizes Natural Language Processing (NLP) to translate the text from source language to target language.
  6. Once translation is obtained, EchoBot uses text-to-speech (TTS) technology to convert the translated text back to spoken words.
  7. Translated speech is then outspoken through the robot's speakers.
